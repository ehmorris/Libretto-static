# Libretto (static)

## Demo
* [ehmorris.github.io/Libretto-static](http://ehmorris.github.io/Libretto-static)

## Development Setup
* Be running ruby 2.1.0 at least
* Run `bundle install`
* Run `middleman server --port 3000`
* To deploy, run `rake publish`
